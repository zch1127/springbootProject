## 安装

这个项目使用 [npm](https://npmjs.com) 和 [maven](https://maven.apache.org) 。请确保你本地安装了它们，并配置了环境变量。

```sh
$ brew install node
$ brew install npm
$ npm  install java
$ npm  install maven
```

## 运行

- 运行`Application.java`类中的Main方法

- 命令启动：mvn spring-boot:run
